요소의 여백(영역) 관련 스타일

    HTML 요소는 총 4가지 영역으로 구성

    1. content
      - 요소의 내용이 작성되어지는 영역(시작 태그와 종료 태그 사이에 작성되는 내용, 값, 이미지 등)
    2. padding
      - 콘텐츠와 보더 사이의 공간
    3. border
      - 요소의 테두리가 지정되는 영역
      - 콘텐츠보다 바깥쪽에서 감싸고 있는 형태
    4. margin
      - 요소와 요소 사이의 간격(여백)을 나타내는 영역

box-sizing

    width/height 크기 설정 시 크기가 적용될 범위를 지정하는 속성

    1. content-box(기본값)
      - width/height 설정 시 콘텐츠 영역의 크기만 설정
    2. border-box
      - width/height 설정 시 콘텐츠 + 패딩 + 보더 크기를 합산한 결과가 되도록 콘텐츠 영역의 크기를 자동으로 조정

방향 테스트

padding, margin, border-width 한 번에 여러 방향 지정

    padding or margin or border-width: 크기
    - 크기 1개: 상하좌우 적용
    - 크기 2개: 상하/좌우 적용
    - 크기 3개: 상/좌우/하 적용
    - 크기 4개: 상/우/하/좌(시계방향) 적용
방향 테스트

margin: auto;

    다른 요소와의 간격을 자동으로 설정하여 가운데 정렬하게 만드는 속성
    - 기본적으로 좌우 간격만 자동으로 조정(상하도 적용하려면 별도의 스타일 적용 필요)
    - inline, inline-block에는 적용 안 됨 -> inline == 텍스트로 취급되기 때문에 텍스트 정령 방법인 text-align 속성을 이용해야 함
